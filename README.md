## 项目简介

本仓库用于备份个人收集的动漫字幕文件，主要特点：

- ✅ 尽可能对所有字幕文件进行字体子集化处理
- ✅ 适配主流压片组发布的视频文件命名
- ✅ 补充部分压片组未附带的字幕文件
- ⚠️ 网络搜集的字幕可能存在错误，请自行判断

## 快速开始

1. 根据你的视频文件来源（VCB-Studio、DBD-Raws 等）找到对应章节
2. 下载对应的字幕压缩包
3. 解压后字幕文件名已适配原始视频命名，直接使用即可

**推荐使用 `00_assfonts` 文件夹内的字幕**：已完成字体子集化，无需额外下载字体文件。

## 命名规范

### 压缩包命名格式

```txt
压片默认文件名.(压片发布UID).字幕组.语言标记
```

### 语言标记说明

| 标记    | 含义       | 示例           |
| ------- | ---------- | -------------- |
| `sc`    | 简体中文   | `xxx.sc.7z`    |
| `tc`    | 繁体中文   | `xxx.tc.7z`    |
| `jp`    | 日文       | `xxx.jp.7z`    |
| `scjp`  | 简日双语   | `xxx.scjp.7z`  |
| `tcjp`  | 繁日双语   | `xxx.tcjp.7z`  |
| `sc&tc` | 简繁双版本 | `xxx.sc&tc.7z` |

### 特殊说明

- 压缩包内文件名已适配资源发布的原始命名
- 自行搜集的字幕已尽量标注字幕组名称，可能存在错误

## 字体子集化说明

`00_assfonts` 文件夹内的字幕使用 [assfonts](https://github.com/wyzdwdz/assfonts) 工具进行字体子集化处理。

**优势：**

- 无需单独下载字体文件
- 字幕文件体积更小
- 兼容性更好

**注意：** 并非所有字幕都经过完整测试，如遇问题可使用非子集化版本。
